# Movie Review Project

## Functionalities
- A user can search any movie
- A user can review a movie
- User Authentication
- The [Movie Database Api](https://developers.themoviedb.org/3/getting-started/introduction) is used

___

To install all dependencies, use :
> npm install

To run app, use :
> node index.js
